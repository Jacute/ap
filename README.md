## Installation

git clone https://github.com/jacute/ap.git

pip install -r requirements.txt


## Dependences for Windows

Codecs
http://www.codecguide.com/download_kl.htm


## Hotkeys

F - open files
Ctrl+F - open directory
Ctrl+C - clear playlist
R - loop
S - mix
Right - forward rewind
Left - back rewind
